dcEmb
=================================

This is the documentation for the `dcEmb <https://github.com/embecosm/dcEmb>`_ 
repository, Embecosm's high performance implementation of Dynamic Causal 
Modelling in C++.

Early Release Warning
#####################
This project is still under active development, and will undergo significant 
changes. Caveat Empor!

Getting Started
###############
See our :ref:`installation instructions <install>` for instructions on
installing the software on your platform. See :ref:`current models 
<currentmodels>` for details on existing DCM models. See
our :ref:`building your own model <howto>` for instructions and a worked 
example of how to implement your own model.
